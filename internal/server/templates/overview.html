📊 Cluster Status Overview

This dashboard provides a real-time view of your PolicyWall admission controller metrics.

ℹ️ Note: Metrics are currently showing mock data. In production, these would be pulled from Prometheus metrics endpoint.

Key Metrics

  • Total Requests: Total number of admission requests processed by PolicyWall
  • Allowed: Requests that passed policy validation
  • Denied: Requests that were rejected by policies
  • Violation Rate: Percentage of requests that violated policies
  • Active Policies: Number of currently enforced admission policies
