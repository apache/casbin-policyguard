{
  "subject": "bob",
  "object": "/api/pods",
  "action": "GET"
}

Denied Request (regular user):

{
  "subject": "bob",
  "object": "/api/secrets",
  "action": "DELETE"
}

Current Policy Rules

The playground uses these default policies:

  • p, admin, /*, * - Admins can access all resources with any action
  • p, user, /api/*, GET - Users can GET resources under /api/
  • g, alice, admin - Alice has admin role
  • g, bob, user - Bob has user role
